#Setpent's Quest

## Overview 
The game's main mechanic is that when your head eats fruit, your body expands. Each fruit has a different effect, and you can collect coins that appear with a very few chance.
The main source of income is advertising, and there are no in-game purchases.
### The more you eat, the stronger you become! Conquer stages with strategic snake action using the effects of various fruits.("먹을수록 강해진다! 다양한 과일의 효과로 스테이지를 정복하는 전략형 스네이크 액션.")

- 먹을수록 변하는 스네이크: 단순히 길어지기만 하는 시대는 끝났습니다. 어떤 과일을 먹느냐에 따라 독특한 특수 효과가 발동됩니다.
- 희귀 코인과 상점: 아주 낮은 확률로 등장하는 코인을 놓치지 마세요! 차곡차곡 모아 생존에 필요한 강력한 아이템을 구매할 수 있습니다.
- 짜릿한 '전액 소비' 시스템: 스테이지의 끝에 도달하면 코인이 초기화됩니다. 인플레이션 걱정 없이, 매 단계 마지막에 모든 코인을 쏟아부어 보상을 획득하세요! (영구스킨조각)
